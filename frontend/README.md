# Schedulr

A powerful and intuitive scheduling platform built for busy professionals.

## Features

- **Smart Scheduling**: Set your availability once and let others book time with you effortlessly.
- **Time Zone Detection**: Automatically detect and convert time zones for seamless global scheduling.
- **Team Coordination**: Schedule meetings with multiple team members simultaneously.
- **Shareable Links**: Generate unique booking links to share with clients and colleagues.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to set up the project locally:

### Prerequisites

Ensure you have Node.js & npm installed.

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd calendly-clone
   ```

3. Install the necessary dependencies:
   ```sh
   npm install
   ```

### Development

Start the development server with auto-reloading:
```sh
npm run dev
```

## Deployment

The project can be built for production using:
```sh
npm run build
```
